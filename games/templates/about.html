About Our Project

About Us

SellGame Online is a web-based game store system created to help users search, view, buy, and manage games easily through the internet.

1. What is your project system?

Our project system is a website for selling games online. Customers can search, view, and purchase different types of games. Users can create accounts, add games to cart, make payments, and view purchase history after buying games.

2. Why did you create this system?

We created this project because many gamers want a simple and fast way to buy games online without going to physical stores. This system is convenient, secure, and user-friendly.

3. Project Completion

The project will be completed by Week 12 and prepared for presentation and demo during Week 12 to Week 14.

Admin Features

The admin dashboard allows the team to manage games, users, orders, purchase history, payment information, and sales records.
